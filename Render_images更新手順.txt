✅ Renderでimagesフォルダを更新するときの正しい手順.txt

【目的】
画像を追加・削除・名前変更したときに、Render上にも正しく反映させる。

【手順】
1. images/ フォルダ内のファイルを普通に操作する（追加・削除・名前変更）

2. 以下のコマンドを順番に実行する：

   git add -A
   git commit -m "全部コミット"
   git push origin main

3. Renderが自動で再デプロイを開始（通常1〜2分かかる）

4. ブラウザで確認（変更が見えない場合は Ctrl + Shift + R で強制リロード）

【注意点】
- git add -A を使うことで「削除」「変更」「追加」をすべてGitが認識する
- 名前変更やファイル削除も、pushしないとRenderには反映されない
- 反映が遅れても焦らず2分ほど待つ
